# Portfolio — Next.js

A modern, animated portfolio website built with Next.js 16, TypeScript, Framer Motion, and Tailwind CSS 4. Showcases skills, projects, and professional experience with a dark theme and smooth animations.

## Features

- **Hero Section** — Animated introduction with typewriter effect and skill role badges
- **The Nexus** — Interactive skill category cards with hover effects
- **Featured Projects** — Dynamic project showcase with filtering
- **About & Contact** — Personal bio with contact links
- **Smooth Animations** — Framer Motion page transitions and scroll-triggered reveals
- **Responsive** — Fully responsive design for all screen sizes

## Tech Stack

- **Framework:** Next.js 16, TypeScript
- **Styling:** Tailwind CSS 4
- **Animation:** Framer Motion
- **Icons:** Lucide React

## Quick Start

```bash
npm install
npm run dev
# Opens at http://localhost:3000
```

## Build

```bash
npm run build
npm start
```
